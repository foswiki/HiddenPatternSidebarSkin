---++ Usage

This is a UI minimisation of Pattern skin to improve _my_ usage on my 1024x768 pixel tablet notebook.

Specifically, it 
   1 reduces the height of the top bar, 
   2 makes the sidebar autohide (or allow the user to hide / pin it and
   3 pins the bottom action bar to the bottom of the browser window, allowing you to always access it wherever on the topic you are.

The core focus is to focus on the current topic, while changing as little of the default skin as possible.

<verbatim>
   * Set SKIN=hiddenpattern,famfamfam,pattern
</verbatim>
